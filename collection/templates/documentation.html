Autenticação

Instruções e Solicitação de Acesso

Esta API utiliza o sistema de autenticação através de TOKEN, assim só é apropriado a utilização com uma conexão cryptografada (HTTPS). Para receber o seu token é precisa contatar o administrador do sistema

Autenticação por Token

Para que a autenticação seja valida, na requisição deve se incluir no header conforme demostrado abaixo, lembrando de deixar os espaços

Authorization: Token 9944b09199c62bcf9418ad846dd0e4bbdfc6ee4b

URLs

URL e Métodos Aceitos
Métodos URLs Descrição
GET POST https://{{ request.get_host }}/book-list GET: Lista de Livros | POST: Novo livro
GET PUT DELETE https://{{ request.get_host }}/book-detail/id GET: Detalhe Livro | PUT: Atualiza Livro | DELETE: Apaga Livro
GET POST https://{{ request.get_host }}/category-list GET: Lista de Categoria | POST: Novo Categoria
GET PUT DELETE https://{{ request.get_host }}/category-detail/id GET: Detalhe Categoria | PUT: Atualiza Categoria | DELETE: Apaga Categoria
GET https://{{ request.get_host }}/collection-detail/id GET: Detalhe Categoria

Corpo das Requisições

POST|PUT: Livros
Atributos Descrição Obrigatório
name Nome do Livro Sim
author Nome do Autor Sim
page_number Quantidade de Páginas Sim
category Categoria do Livro id das categorias disponíveis Sim
POST|PUT: Categorias
Atributos Descrição Obrigatório
name Nome da Categoria Sim

Filtros

Filtros para categorias
